Política de Privacidad

Esta aplicación utiliza la API de Facebook únicamente para publicar contenido en páginas administradas por el usuario. No se recopila información personal para otros fines, y todos los datos se manejan conforme a las políticas de Facebook.

Información que recopilamos

Solo se accede a la información necesaria para publicar en tus páginas (como el ID de la página y permisos de publicación). No se comparte con terceros.

Uso de la información

La información se utiliza exclusivamente para permitir que la aplicación publique contenido en tus páginas de Facebook o Instagram según tu autorización.
